# dirty-markup-cli

--------

## Installation:

`npm i -g dirty-markup-cli`

## Usage:

```shell
  dirty-markup-cli

  usage:
  cat index.html | dirty-markup
  dirty-markup --css --4 < styles.css

  options:
  mode: --html | --css | --js (default: html)
  indent: --2 | --4 | --8 | --tabs (default: 2)
```
